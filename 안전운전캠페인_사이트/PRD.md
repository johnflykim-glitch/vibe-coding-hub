## 1. 제품 개요

단일 HTML 파일(`index.html`)로 구성된 1-page 정적 사이트.
외부 JS 프레임워크 없이 순수 HTML + CSS + 인라인 JavaScript로 작성되었으며,
파일을 브라우저로 바로 열거나 정적 호스팅 서버에 올려 사용한다.

## 2. 디자인 시스템

### 컬러
| 변수 | 값 | 용도 |
|---|---|---|
| `--navy` | `#003087` | 헤더, 푸터, 섹션 제목 배경 |
| `--blue` | `#0050b3` | 카드 상단 라인, 점, 레이블 |
| `--sky` | `#1976d2` | 히어로 그라디언트 중간 |
| `--yellow` | `#f47920` | 포인트 컬러 (주황), 슬로건 강조, 구분선 |
| `--light` | `#f5f7fa` | 페이지 배경, QR 박스 배경 |
| `--gray` | `#78909c` | 본문 서브텍스트 |

### 폰트
- **Noto Sans KR** (Google Fonts CDN)
- weight: 300 / 400 / 500 / 700 / 900

### 히어로 그라디언트
```css
background: linear-gradient(135deg, #003087 0%, #0050b3 55%, #c85a00 100%);
```

### 도로 구분선 패턴
```css
background: repeating-linear-gradient(
  90deg,
  #f47920 0px, #f47920 40px,
  #003087 40px, #003087 70px
);
```

## 4. 컴포넌트 명세

### 4-1. Top Bar
- 배경: `--navy`
- 좌측: 로고 아이콘(🛣️) + "한국도로공사" 텍스트
- 우측: 오늘 날짜 (JavaScript `Date` 객체로 동적 렌더링) + 뱃지
- 날짜 형식: `YYYY년 M월 D일 (요일)` — 페이지 로드 시 자동 갱신

### 4-2. Hero Section
- 레이블: `VIBE CODING COMPETITION 2026` (영문 소문자, 테두리 pill)
- 슬로건: `AI와 함께, 도로의 미래를 코딩으로 설계하다`
  - 강조 단어("코딩")는 `<em>` 태그 + `--yellow` 색상
- 서브텍스트: 소속 부서명 포함

### 4-3. 핵심 안내 카드 (3개)
카드마다 공통 구조:
```
아이콘 (이모지)
POINT 0X  ← 섹션 번호 레이블
제목 (bold, --navy)
설명 (13px, --gray, 부서명 <strong> 강조 포함)
```
- 카드 상단에 `border-top: 4px solid --blue`
- hover 시 `translateY(-4px)` + 그림자 강화

| 카드 | 아이콘 | 제목 |
|---|---|---|
| 1 | 💡 | 누구나 참가 가능, AI가 파트너 |
| 2 | 🛣️ | 주제 — 도로·교통 문제 해결 |
| 3 | 🏆 | 총 상금 500만 원 시상 |

### 4-4. 대회 일정 박스
점(dot) + 항목명 + 날짜 리스트 형태.

| 항목 | 일정 |
|---|---|
| 참가 신청 | 2026.05.19(화) ~ 06.06(금) |
| AI 툴 워크숍 | 2026.06.03(화) 온라인 |
| 개발 기간 | 2026.06.09(월) ~ 06.27(금) |
| 발표 심사 | 2026.07.04(금) 본사 대강당 |

하단에 주황 left-border 하이라이트 박스:
> 🎯 사전 AI 툴 활용 워크숍 — 6월 3일(화) 온라인 진행

### 4-5. QR 코드 자리
- 130×130px 점선 박스 (dashed border)
- "▦ QR 코드 삽입 위치" 텍스트 플레이스홀더
- 실제 QR 이미지 삽입 시 `<img>` 태그로 교체

### 4-6. Footer
- 배경: `--navy`
- 내용: 기관명 | 담당 부서 | 문의 이메일 | 저작권
