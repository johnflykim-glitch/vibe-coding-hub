# BRD: 한국도로공사 교통량 모니터링 대시보드

**문서 버전**: 1.0  
**작성일**: 2026-05-11  
**대상 독자**: 개발자, Claude Code  

---

## 1. 프로젝트 개요

### 1.1 배경
한국도로공사는 전국 고속도로의 교통량과 속도 데이터를 수집하고 있다. 이 데이터를 직관적으로 시각화하여 교통 상황을 실시간으로 모니터링하고, 급증 구간이나 정체 구간을 빠르게 식별할 수 있는 도구가 필요했다.

### 1.2 목적
- 5개 고속도로 노선의 교통량·속도 데이터를 단일 화면에서 확인
- 시간대별·차종별·노선별 필터링을 통한 심층 분석
- 이상 징후(교통량 급증, 극심 정체) 자동 감지 및 경고

### 1.3 범위
- **데이터 기간**: 2026-04-01 ~ 2026-04-07 (7일)
- **대상 노선**: 경부, 남해, 서해안, 영동, 중부 고속도로
- **차종**: 승용차, 소형화물, 중형화물, 대형화물, 버스
- **시간 단위**: 1시간 단위 (00시~23시)

## 3. 비즈니스 요구사항

### BR-01: 통합 교통량 현황판
- 전체 노선 또는 특정 노선의 교통량을 한 화면에서 확인할 수 있어야 한다.
- 총 교통량, 평균 속도, 피크 시간대, 최저 속도 시간대를 즉시 파악할 수 있어야 한다.

### BR-02: 다차원 필터링
- 노선, 시간대, 차종 3개 축으로 데이터를 필터링할 수 있어야 한다.
- 필터 변경 즉시 모든 지표와 차트가 갱신되어야 한다.

### BR-03: 시각적 데이터 분석
- 시간대별 교통량 분포를 통해 피크 타임을 시각적으로 확인할 수 있어야 한다.
- 차종 구성비, 요일별 추이, 노선별 속도 비교를 차트로 제공해야 한다.

### BR-04: 이상 구간 자동 경고
- 전 시간 대비 교통량이 급증한 구간을 자동으로 감지해야 한다.
- 평균속도가 임계값 이하인 극심 정체 구간을 경고해야 한다.
- 경고는 심각도에 따라 분류되어 표시되어야 한다.

### BR-05: 정체 순위 파악
- 속도가 가장 낮은 상위 5개 구간(노선+날짜+시간대 조합)을 순위표로 제공해야 한다.

### BR-06: 독립 실행
- 서버나 외부 DB 없이 브라우저에서 단독으로 실행되어야 한다.
- HTML 파일 하나를 열면 모든 기능이 즉시 동작해야 한다.

## 4. 핵심 비즈니스 규칙

| 규칙 ID | 내용 |
|---------|------|
| BIZ-01 | 교통량 단위는 "대(臺)" |
| BIZ-02 | 속도 단위는 km/h |
| BIZ-03 | 경고 등급 CRITICAL: 전 시간 대비 +60% 이상 급증 OR 평균속도 < 45 km/h |
| BIZ-04 | 경고 등급 WARNING: 전 시간 대비 +35% 이상 ~ +60% 미만 급증 |
| BIZ-05 | 경고는 최대 12개까지 표시, CRITICAL을 WARNING보다 우선 표시 |
| BIZ-06 | Top 5 정체 순위는 필터와 무관하게 항상 전체 RAW 데이터 기준으로 산출 |
| BIZ-07 | 속도 색상 기준: 빨강 < 50 km/h, 주황 50~70 km/h, 초록 > 70 km/h |

## 5. 데이터 정의

### 5.1 원시 데이터 구조
```
RAW 배열: [[날짜, 노선, 시간대, 차종, 교통량, 속도], ...]
```

| 인덱스 | 필드 | 타입 | 예시 |
|--------|------|------|------|
| r[0] | 날짜 | String | "2026-04-01" |
| r[1] | 노선 | String | "경부고속도로" |
| r[2] | 시간대 | String | "00시" ~ "23시" |
| r[3] | 차종 | String | "승용차" |
| r[4] | 교통량 | Number | 516 |
| r[5] | 속도 | Number | 75 |

### 5.2 데이터 규모
- **총 레코드**: 5노선 × 7일 × 24시간 × 5차종 = **4,200개**
- **저장 방식**: HTML 파일 내 JavaScript 배열로 하드코딩

### 5.3 참조 상수
```javascript
ROUTES:    ['경부고속도로', '남해고속도로', '서해안고속도로', '영동고속도로', '중부고속도로']
TYPES:     ['승용차', '소형화물', '중형화물', '대형화물', '버스']
HOURS:     ['00시', '01시', ..., '23시']  // 24개
DATE_LIST: ['2026-04-01', ..., '2026-04-07']  // 7일
DAYS:      ['수', '목', '금', '토', '일', '월', '화']  // DATE_LIST와 1:1 대응
```

## 6. 제약 조건

- **기술 제약**: Chart.js CDN 필요 (오프라인 불가)
- **데이터 제약**: 정적 데이터 (실시간 갱신 없음)
- **브라우저 제약**: 최신 브라우저 권장 (ES6+ 문법 사용)
- **화면 제약**: 최소 900px 이상 권장 (모바일 부분 지원)
